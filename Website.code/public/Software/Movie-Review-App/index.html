This project requires Bun and cannot run directly as a static HTML page. To view this project:

Step 1: Download the Project

Option A: Download just this project folder (recommended):

# Using SVN to download just this folder
svn export https://github.com/VoidLance/Course-Files/trunk/JavaScript/React/movie-review-app
cd movie-review-app

Option B: Clone the entire repository:

git clone https://github.com/VoidLance/Course-Files.git
cd Course-Files/JavaScript/React/movie-review-app

Or download manually from: GitHub Repository

Step 2: Install & Run Locally

bun install
bun run dev

Then open http://localhost:3000 in your browser

Requirements:

  • Bun (latest version)
  • SVN (for Option A) - usually pre-installed on Mac/Linux, or Git only (for Option B)
  • TMDB API key (configured in .env.local)
